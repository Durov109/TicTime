Файл  code_v1.0.py  является первой версией (не рекомендуется запускать)
Файл test.py  является тестовой программой взятой из документации
Файл requirements.txt содержит библиотеки и их версии

Рабочим файлом является tic_time.py, он сохраняет время в Timer и высчитывает его в Total. Программа работает на python
 3.11 версии. (На других не пробовал)
___________________________________________________________________
Перед запуском нужно либо активировать виртуальное окружение :

активация окружения происходит так: venv\Scripts\activate   или .\venv\Scripts\activate
после вы должны увидеть синие слова в терминале (venv), это означает что окружение активировано
и вы можете запускать программу.

деактивация окружения происходит так: deactivate


Или самому скачать библиотеку PySimpleGUI==4.60.5 :
pip install pysimplegui
